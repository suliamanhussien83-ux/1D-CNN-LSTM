# 1D-CNN-LSTM
Complete R implementation accompanying the study on forecasting the Iraqi fiscal surplus/deficit using oil prices. The repository includes embedded data, ARDL and NARDL estimations, deep learning models (LSTM, 1D-CNN, and Hybrid 1D-CNN-LSTM), Bayesian hyperparameter optimization, forecasting evaluation, and Diebold–Mariano tests.
# Predicting Public Budget Surplus and Deficit Using a Hybrid 1D-CNN–LSTM Model

This repository contains the complete R implementation accompanying the study on forecasting the Iraqi fiscal surplus/deficit using oil prices.

## Models
- ARDL
- NARDL
- LSTM
- 1D-CNN
- Hybrid 1D-CNN-LSTM

## Data
Monthly data for the Iraqi economy covering the period 2008-2025.

## Evaluation Measures
- RMSE
- MAE
- Directional Accuracy (DA)
- Diebold–Mariano Test
